CaPybara Documentation
======================

**CaPybara** is a data-driven Python library designed to streamline your entire data workflow. Whether you're wrangling messy datasets, uncovering insights, or building predictive models, CaPybara provides high-level abstractions that let you focus on what matters: solving problems with data.
