## Why CaPybara?

Modern data work often means juggling multiple libraries, each with its own learning curve and quirks. CaPybara consolidates the most common data operations into a unified interface, reducing cognitive overhead and letting you move from idea to insight faster.

Built with developers in mind, CaPybara prioritizes:
- **Clarity over cleverness** - APIs that make sense on first read
- **Pragmatism over purity** - Solutions that work in the real world
- **Flexibility** - Play well with the Python data ecosystem you already know
